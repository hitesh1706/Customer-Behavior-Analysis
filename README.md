# Customer Behavior Analysis

## Overview

This project is based on customer shopping behavior analysis using Python, SQL, and Power BI. The main objective was to clean and transform raw customer data, analyze customer purchasing patterns, and create an interactive dashboard to understand business performance and customer trends.

The project follows a complete data analytics workflow starting from data cleaning in Python, storing cleaned data in MySQL, performing business analysis using SQL, and finally building a dashboard in Power BI.

---

# Tools Used

* Python
* Pandas
* NumPy
* MySQL
* SQLAlchemy
* Power BI

---

# Python Data Cleaning & Processing

Data cleaning and preprocessing were done using Python.

### Work Performed

* Imported and explored raw dataset
* Handled missing values using median values
* Standardized column names
* Created new columns for analysis
* Created age groups
* Converted purchase frequency into days
* Connected Python with MySQL
* Exported cleaned dataset into SQL database

### Python Concepts Used

* Pandas DataFrames
* NumPy
* Lambda Functions
* GroupBy Operations
* Feature Engineering
* Database Connection using SQLAlchemy

---

# SQL Analysis

Business-related questions were solved using SQL queries.

### SQL Concepts Used

* Aggregate Functions
* GROUP BY
* CASE Statements
* Common Table Expressions (CTE)
* Window Functions
* Subqueries

### Business Insights Generated

* Revenue generated by each gender
* Customers purchasing with discounts
* Top-rated products
* Subscription spending analysis
* Repeat customer analysis
* Revenue by age group
* Product ranking within categories

---

# Power BI Dashboard

An interactive dashboard was created in Power BI to visualize customer behavior and business insights.

### Dashboard Includes

* Total Revenue
* Total Customers
* Average Purchase
* Average Rating
* Revenue by Category
* Subscription Analysis
* Payment Method Analysis
* Product Performance
* Interactive Filters and Slicers

---

# Key Findings

* Clothing category generated the highest revenue
* Subscription customers contributed higher spending
* Young adults were the most active buyers
* Discount campaigns influenced customer purchases
* Certain payment methods were used more frequently than others
